# Projeto Olá

## Descrição

Este é um projeto HTML5 simples e puro, sem dependências externas. O objetivo é demonstrar uma página web básica com um título "Olá" e um parágrafo de exemplo.

## Como visualizar

1. Faça o download ou clone este repositório.
2. Navegue até a pasta `projeto-ola`.
3. Abra o arquivo `index.html` em qualquer navegador web moderno (Google Chrome, Firefox, Edge, Safari, etc.).

Não é necessário instalar nenhuma dependência ou servidor local. O arquivo HTML é auto-contido e pode ser aberto diretamente.
